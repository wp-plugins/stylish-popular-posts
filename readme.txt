Stylish Popular Posts creates a widget which displays popular posts based on the number of comments.

== Description ==
This is a very simple, easy to use plugin which creates a widget that can be used to display popular posts with thumbnails based on the number of comments. Users can choose how many posts they want to show in the widget settings. 

= Requirements =
Your Theme must support widgets in order to use this plugin. Visit [WordPress Widgets](http://codex.wordpress.org/WordPress_Widgets) for more details.

= Regenerate Thumbnails =
This plugin uses custom thumbnails, you can use [Regenerate Thumbnails](http://wordpress.org/plugins/regenerate-thumbnails/) to regenerate the thumbnails for your image attachments.

= Translations =
To translate the plugin to a particular language, use the file provided in /languages/ folder.

== Installation ==

= From your WordPress dashboard =
1. Visit 'Plugins > Add New'
2. Search for 'stylish popular posts'
3. Activate Simple music widget from your Plugins page.
4. Go to Appearance > Widgets
5. Drag and drop 'stylish popular posts' in the widget area
6. Configure the widget

= From WordPress.org =
1. Download and upload the 'stylish-popular-posts' directory to /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets
4. Drag and drop 'stylish popular posts' in the widget area
5. Configure the widget

== Frequently Asked Questions == 

= Can I show popular posts based on the number of views? =
No, the plugin only shows popular posts based on the number of comments submitted. 

= This doesn't look good on my Theme =
If the current style does not match your Theme, you can modify ../stylish-popular-posts/css/style.css
